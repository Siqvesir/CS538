Environment Requirments for CS538 Project

To Use in a Terminal:
1) Download DES_Atk.py.py (Note: Named with a .py in Windows as a python file caused the extra .py, will not run properly without .py.py)
2) Ensure the Windows or Linux terminal have Python readily available
3) Open the terminal from the location that DES_Atk.py.py is saved
4) Use the command "python3 DES_Atk.py.py" in the cmd line
5) Follow the User-input directions and then wait for it to run
*The code takes 14+ minutes to run on my decently high-speced system, it may take longer or shorter on other systems*
